# 🤖 AI Chatbot using LangChain, OpenAI API & Gradio

This project implements an AI-powered conversational chatbot using LangChain and OpenAI’s GPT-3.5-turbo model.

## 📌 Project Overview
The goal of this project is to build an intelligent chatbot capable of generating human-like responses.

## ⚙️ Features
- AI-powered conversational chatbot  
- OpenAI GPT-3.5-turbo integration  
- LangChain framework  
- Gradio web interface  

## 🛠️ Technologies Used
- Python  
- LangChain  
- OpenAI API  
- Gradio  
- Python-dotenv  

## 🚀 How to Run
1. Clone repository  
2. Install dependencies  
3. Add `.env` file  
4. Run `python chatbot.py`

## 📌 Conclusion
This project demonstrates real-world integration of LLMs using LangChain and OpenAI API.
